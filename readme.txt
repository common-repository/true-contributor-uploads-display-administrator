This Plugin is only the administrator to be able to see the images uploaded by others.

== Description ==

This plugin allows contributors to upload images, and also allows only administrators to see images uploaded by others.

Contributors can only manage their own images.

== Installation ==

1. Upload the unzipped plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. That's all! No configuration necessary.
